# Security Policy

## Reporting a Vulnerability

obdex is a public dataset and a small set of build tools. The dataset itself has no security implications; the tooling runs locally or in CI and processes only files in this repository.

If you discover a security issue in the build/validation tooling, the GitHub Actions workflows, or any infrastructure linked from this project, **please do not open a public issue**. Instead, report it privately to:

**z@xdr5.de**

You will receive an acknowledgement within 7 days. Coordinated disclosure is appreciated for any non-trivial issue before public discussion.

## Out of Scope

- Bugs, errors, or imprecisions in the data — please open a [Correction issue](https://github.com/foerbsnavi/obdex/issues/new?template=correction.yml) instead.
- Missing codes — open a [New code issue](https://github.com/foerbsnavi/obdex/issues/new?template=new-code.yml).
- Style or wording suggestions — pull request preferred.
